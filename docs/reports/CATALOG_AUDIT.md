### ❌ Проблема 1.1: Відсутнє поле для завантаження зображення категорії
**Опис:** Незважаючи на наявність поля `imageUrl: String?` в моделі [`Category`](prisma/schema.prisma:191), у формі створення/редагування категорій відсутній компонент для завантаження зображення.

**Вплив:** Користувачі не можуть додавати зображення до категорій, що погіршує візуальну навігацію на вітрилі.

**Рішення:** Додати компонент `ImageUpload` у [`CategoriesClient`](src/app/dashboard/[storeId]/categories/client.tsx:238) аналогічно до продуктів.

### ❌ Проблема 2.1: Відсутність Drag-and-Drop сортування фото
**Опис:** Користувачі не можуть змінити порядок світлинь в товарі шляхом перетягування.

**Вплив:** Неможливість налаштувати бажаний порядок відображення фото (перше фото — головне).

**Рішення:** Інтегрувати бібліотеку `@dnd-kit` або `react-beautiful-dnd` в компонент завантаження зобраень.

### ❌ Проблема 2.2: Відсутність Alt-тексту для зобраень
**Опис:** Немає поля для введення альтернативного тексту, що важливо для SEO та доступності.

**Вплив:** Погіршення SEO-оптимізації та доступності для людей з вадами зору.

**Рішення:** Додати поле `altText` до структури зобраення або як окреме поле товару.

### ❌ Проблема 2.3: Відсутність мультимовних вкладок (Tabs)
**Опис:** Незважаючи на наявність JSON поля `translations`, в UI відсутні вкладки для перемикання між мовами (UK, EN, PL, RU). Є лише кнопка AI-перекладу.

**Вплив:** Користувачі не можуть вручну відредагувати переклад після AI-генерації або створити власний переклад без AI.

**Рішення:** Додати Tabs-компонент з окремими вкладками для кожної мови з полями: Назва, Опис, SEO Title, SEO Description.

## 📝 Додаткові нотатки

1. **Multi-tenant ізоляція:** Всі таблиці (`Category`, `Product`, `ProductVariant`) мають поле `storeId`, що забезпечує правильну ізоляцію даних між орендарями.

2. **ProductVariant модель:** Модель [`ProductVariant`](prisma/schema.prisma:864) реалізована правильно з полями `name`, `sku`, `price`, `stock`, `options` (JSON).

3. **Translations поле:** JSON поле `translations` в [`Product`](prisma/schema.prisma:224) та [`Category`](prisma/schema.prisma:195) дозволяє зберігати переклади, але потребує кращого UI.
